## 下载 nvm-windows

- https://github.com/coreybutler/nvm-windows/releases
- https://github.com/coreybutler/nvm-windows/releases/download/1.2.2/nvm-setup.exe

## 安装 nvm

- Node.js symlink： D:\apps\nodejs
- NVM root： D:\apps\nvm
- ⚠ 安装完成后 关闭并重新打开终端

## 安装 Node（推荐 LTS）

```cmd
nvm install 22
nvm use 22

node -v
npm -v
```
